# Sushi Shop — Магазин доставки роллов 🍣

Простой и удобный интерфейс для выбора и заказа роллов. Проект реализован на чистом JavaScript (Vanilla JS) с использованием Bootstrap для верстки.

## Функционал
* **Каталог товаров:** Список популярных роллов с описанием веса и количества.
* **Интерактивный счетчик:** Возможность изменять количество товара перед добавлением.
* **Корзина:** * Добавление товаров в корзину без перезагрузки страницы.
    * Автоматическое суммирование одинаковых позиций.
    * Подсчет общей стоимости (в процессе реализации).
* **Оформление заказа:** Простая форма для ввода контактных данных.

## Технологии
* **HTML5 & CSS3** (Flexbox, CSS Grid)
* **Bootstrap 4** — для адаптивной сетки и базовых компонентов.
* **JavaScript** — делегирование событий, манипуляции с DOM, работа с шаблонами строк.
