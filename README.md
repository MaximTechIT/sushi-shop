🍣 Sushi Shop — Delivery Service App
Проект простого и удобного интерфейса для выбора и заказа роллов. Реализован на чистом JavaScript (Vanilla JS) с использованием Bootstrap 4 для быстрой и адаптивной верстки.

🚀 Функционал
Динамический каталог: Список популярных позиций с отображением веса, цены и состава.

Интерактивный счетчик: Изменение количества товара как в основном каталоге, так и внутри корзины.

Умная корзина:

Добавление товаров без перезагрузки страницы (DOM-манипуляции).

Автоматическое суммирование одинаковых позиций при повторном добавлении.

Live-расчет: Мгновенный пересчет общей стоимости заказа и стоимости доставки.

Динамическое отображение: форма заказа и статус «Корзина пуста» переключаются автоматически.

Оформление заказа: Интуитивно понятная форма для ввода контактных данных.

🛠 Технологии
Frontend: HTML5, CSS3 (Flexbox, Grid)

UI Framework: Bootstrap 4

Logic: JavaScript (ES6+)

Делегирование событий (Event Delegation).

Работа с шаблонами строк (Template Literals).

Манипуляция узлами DOM.

📂 Структура проекта
Логика разделена на модули для удобства поддержки:

counter.js — управление счетчиками количества товара.

cart.js — логика добавления товаров в корзину и создания HTML-шаблонов.

toggleCartStatus.js — контроль состояния интерфейса (пустая/полная корзина).

calcCartPrice.js — математический модуль для расчета итоговой суммы и условий доставки.
